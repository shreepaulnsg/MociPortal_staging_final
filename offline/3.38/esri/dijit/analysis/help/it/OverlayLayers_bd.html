Sovrapponi Layer

Sovrapponi layer consente di combinare due o più layer in un unico layer. La sovrapposizione è un processo analogo all'analisi di un insieme di mappe ed alla creazione di un unica mappa contenente tutte le informazioni trovate nel gruppo. La sovrapposizione non è una semplice unione di linee; nel prodotto finale vengono infatti inclusi tutti gli attributi delle feature della sovrapposizione. La sovrapposizione costituisce la risposta ad una delle più elementari domande di geografia, ovvero "Cosa sta sopra e cosa sta sotto?"

Di seguito sono riportati degli esempi:

  • Quali lotti sono compresi nella pianura alluvionale da 100 anni?
  • Quale uso del suolo si fa di un determinato tipo di terreno?
  • Quali pozzi si trovano in basi militari abbandonate?

Se l'opzione Usa estensione mappa corrente è selezionata, verranno sovrapposte solo le feature nei layer di input e di sovrapposizione visibili nell'estensione mappa corrente. Se non è selezionata, verranno sovrapposte tutte le feature sia nel layer di input che in quello di sovrapposizione, anche se esterne all'estensione mappa corrente.

Scegli il layer di input

Il layer di punti, linea o area che verrà sovrascritto.

Oltre a scegliere un layer dalla mappa, è possibile selezionare Scegli layer di analisi in fondo all'elenco a discesa. Con questa opzione si apre una galleria in cui è possibile individuare e selezionare dataset di varie fonti da utilizzare nell'analisi.

Scegliere il layer per la sovrapposizione

Il layer da sovrapporre sul layer di input.

Oltre a scegliere un layer dalla mappa, è possibile selezionare Scegli layer di analisi alla base dell'elenco a discesa per cercare un dataset di condivisione file Big Data o un feature layer. In via facoltativa, è possibile applicare un filtro al layer di input o applicare una selezione sul layer hosted aggiunto alla mappa. I filtri e le selezioni vengono applicati solo per l'analisi.

Scegliere il metodo di sovrapposizione

Specifica il metodo di sovrapposizione che combinerà il layer di input e il layer di sovrapposizione.

  • Incrocia: si combineranno solo le feature o le parti di feature del layer di sovrapposizione che risultano sovrapposte a feature del layer di input.
  • Cancella: si combineranno le feature o le parti di feature del layer di sovrapposizione che non risultano sovrapposte ad aree del layer di input.
  • Unione: si combineranno tutte le feature di entrambi i layer, creando un'unione geometrica del layer di input e del layer di sovrapposizione. Si scriveranno sul layer tutte le feature e tutti i loro attributi.
  • Identità: si combineranno le feature o le parti di feature del layer di input e del layer di sovrapposizione. Le feature o le porzioni di feature che si sovrappongono tanto nel layer di input quanto nel layer di sovrapposizione verranno scritte sul layer di output.
  • Differenza simmetrica: si combineranno le feature o le parti di feature del layer di input e del layer di sovrapposizione che non si sovrappongono.

Nome del layer dei risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il tipo di feature, ovvero area, linea o punto, è uguale a quello di input e sovrapposizione. Il nome predefinito è basato sul metodo di sovrapposizione e sul nome del layer di input. Se il layer esiste già, verrà chiesto di fornire un altro nome.

Utilizzare la casella a discesa Salva risultato in per specificare il nome di una cartella di I miei contenuti in cui salvare il risultato.
