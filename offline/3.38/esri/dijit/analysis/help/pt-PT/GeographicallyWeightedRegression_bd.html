Regressão Ponderada Geograficamente

Efetua a Regressão Ponderada Geograficamente (GWR), que é um formulário local de regressão linear utilizado para modelar relações espacialmente variáveis.

Escolher uma camada para analisar

A camada que contém elementos de pontos com as variáveis dependentes e explanatórias.

Para além de escolher uma camada do seu mapa, pode escolher Escolher Camada de Análise na parte inferior da lista pendente para navegar até aos seus conteúdos de um conjunto de dados de partilha de ficheiros de big data ou camada de elementos. Opcionalmente, pode aplicar um filtro na sua camada de entrada ou aplicar uma seleção em camadas alojadas adicionadas ao seu mapa. Os filtros e seleções são aplicados apenas para análise.

Escolher o campo a modelar

O campo numérico que contém os valores observados a modelar e o tipo de valor que está a modelar.

Escolher os campos explanatórios

Um ou mais campos que representam as variáveis explanatórias (campos) que ajudam a prever o valor. Apenas os campos numéricos serão visíveis.

Escolher como a vizinhança será determinada

Especifica se a vizinhança utilizada é construída como uma distância fixa ou se pode variar numa extensão espacial dependendo da densidade dos elementos.

  • Número de Vizinhos — O tamanho da vizinhança é uma função de um número especificado de vizinhos incluído nos cálculos para cada elemento. Onde os elementos forem densos, a extensão espacial da vizinhança é mais pequena; onde os elementos estiverem dispersos, a extensão espacial da vizinhança é maior. Quando escolher esta opção, selecione o número de vizinhos que pretende incluir. Deve ser um número inteiro entre 2 e 5000.
  • Faixa de Distância — O tamanho da vizinhança é uma distância constante ou fixa para cada elemento. Quando escolher esta opção, selecione a faixa de distância para representar a extensão espacial da vizinhança.

Escolher como os elementos vizinhos são ponderados

Especifica o tipo de kernel que será utilizado para fornecer a ponderação espacial no modelo. Kernel define a forma como o elemento se relaciona com outros elementos na sua vizinhança.

  • Biquadrado — Será atribuído um valor de 0 a qualquer elemento fora da vizinhança especificada. Isto é por omissão.
  • Gaussiano — Todos os elementos receberão valores, mas estes irão tornar-se exponencialmente mais pequenos quanto mais afastados estiverem do elemento-alvo.

Nome da camada resultante

O nome da camada que será criada. Se estiver a escrever para ArcGIS Data Store, os seus resultados serão guardados em O Meu Conteúdo e adicionados ao mapa. Se estiver a escrever para uma partilha de ficheiros big data, os seus resultados serão guardados na partilha de ficheiros big data e adicionados ao seu manifesto. Não serão adicionados ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Caso a camada já existe, a ferramenta irá falhar.

Ao escrever para ArcGIS Data Store (armazenamento relacional ou espaciotemporal de dados big data) utilizando a caixa suspensa Guardar resultado em pode especificar o nome de uma pasta em O Meu Conteúdo, onde será armazenado o resultado.
