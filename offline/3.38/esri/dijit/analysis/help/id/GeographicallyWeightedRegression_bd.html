Regresi Berbobot secara Geografis

Lakukan Regresi Berbobot secara Geografis (GWR), yang merupakan bentuk lokal dari regresi linier yang digunakan untuk memodelkan hubungan yang bervariasi secara spasial.

Pilih layer yang akan dianalisis

Layer berisi fitur titik yang berisi variabel terikat dan penjelas.

Selain memilih layer dari peta, Anda dapat memilih Pilih Layer Analisis di bawah daftar tarik-turun untuk menelusuri konten Anda untuk set data atau feature layer berbagi file big data. Anda dapat secara opsional menerapkan filter pada layer input Anda atau menerapkan pilihan pada layer yang di-host yang ditambahkan ke peta Anda. Filter dan pilihan hanya diterapakan untuk analisis.

Pilih kolom untuk membuat model

Kolom numerik berisi nilai-nilai yang diamati untuk dijadikan model dan jenis nilai yang Anda jadikan model.

Pilih kolom penjelas

Satu kolom atau lebih mewakili variabel penjelas (kolom) yang membantu memprediksikan nilai. Hanya kolom numerik yang akan terlihat.

Pilih bagaimana lingkungan akan ditentukan

Menentukan apakah lingkungan yang digunakan dibangun sebagai jarak tetap atau diizinkan bervariasi dalam tingkat spasial tergantung pada kepadatan fitur.

  • Jumlah Lingkungan—Ukuran lingkungan adalah fungsi dari sejumlah tetangga yang dimasukkan dalam perhitungan untuk setiap fitur. Di mana fitur padat, tingkat spasial lingkungan lebih kecil; di mana fitur jarang, tingkat spasial lingkungan lebih besar. Saat Anda memilih opsi ini, pilih jumlah tetangga yang ingin Anda sertakan. Angka harus bilangan bulat antara 2 dan 5000.
  • Pita Jarak—Ukuran lingkungan adalah jarak konstan atau tetap untuk setiap fitur. Saat Anda memilih opsi ini, pilih pita jarak untuk mewakili tingkat spasial dari lingkungan tersebut.

Pilih bagaimana fitur lingkungan berbobot

Menentukan tipe kernel yang akan digunakan untuk memberikan pembobotan spasial dalam model. Kernel menentukan bagaimana setiap fitur terkait dengan fitur-fitur lain dalam lingkungannya.

  • Bisquare—Berat 0 akan diberikan ke fitur apa pun di luar lingkungan yang ditentukan. Ini adalah default.
  • Gaussian—Semua fitur akan menerima bobot, tetapi bobot menjadi lebih kecil secara eksponensial semakin jauh dari fitur target.

Nama layer hasil

Nama layer yang akan dibuat. Jika Anda menulis ke sebuah ArcGIS Data Store, hasil Anda akan disimpan di Konten Saya dan ditambahkan ke dalam peta. Jika Anda menulis ke berbagi file big data, hasil Anda akan disimpan ke berbagi file big data dan ditambahkan ke manifestasinya. Ini tidak akan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer sudah ada, alat tidak akan berfungsi.

Saat menulis ke ArcGIS Data Store (penyimpanan big data relasional atau spasiotemporal) menggunakan Simpan hasil di kotak tarik-turun, Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan.
